## Explanation / Breakdown

### Stepwise 12-Lead Sequence

1. **Rate**: Fast, slow, regularity
2. **Rhythm**: Origin/P-wave, axis deviation
3. **Intervals**: PR, QRS, QTc
4. **QRS Locks**: LBBB, RBBB, IVCD
5. **Chamber Hypertrophy**: LVH, LAE, RAE, RVH
6. **ST/T Segments**: Elevation/depression, T-wave inversion (Wellen’s, ischemia)
7. **Localization**: Identify regions (II/III/aVF = inferior, V1–V4 = anterior, V5–V6 = lateral, V7–V9 = posterior)
8. **Reciprocal Changes**
9. **Mimics/Confounders**: HyperK, Early repol, Pericarditis, Brugada

***

## Common Mistakes

- Relying on machine readout or snapshot only.
- Ignoring axis deviation and subtle T-wave changes (ACS can hide).
- Misreading RBBB with anterior STEMI.
- Not using posterior and right-sided leads in the right context.

***

## Clinical Pearls

- Always run V4R for inferior MI/RV involvement.
- T-wave inversion in aVL = subtle inferior ischemia.
- De Winter’s T-waves: Proximal LAD occlusion without classic ST elevation.
- Wellen’s: Deep T-wave inversion V2/3 = critical LAD.
- Posterior MI: Depressed V1–V3; run V7–V9 for ST elevation.

***

## Decision Trees

**12-Lead Emergent Action Sequence**

1. ACUTE? STEMI/HyperK/Brugada?
    - If yes: Activate protocol, immediate consult
2. Localize: Inferior, Anterior, Lateral, Posterior, RV
3. Axis/chamber/interval red flags?
    - Correct confounders (lytes, meds, bundle, etc.)
4. Serial changes? Can’t-miss pattern?
